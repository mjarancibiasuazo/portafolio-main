Portafolio

Mis mejores proyectos



Proyecto Futbolito

Proyecto de titulo de aplicación móvil que organiza torneos de baby Futbol. En este proyecto usamos Expo para usar la galeria de imagenes del móvil, también utilzamos la Api de Google para geolocalizar las canchas de los torneos.

JavaScript React Native Expo Firebase API Rest CSS3


Planificador de Gastos y Presupuestos

Proyecto de planificación de gastos y presupuestos personal. En este proyecto más complejo utilizamos los hooks de estados useReducer y useContext. Esta diseñado con el framework de CSS3 Tailwind y para el gráfico de gastos usamos la libreria de React.js circular progressbar .

TypeScript Vite React.JS Node.JS Tailwind



Proyecto Administrador de Pacientes

Proyecto de registro de pacientes de una veterinaria en donde podemos agregar, editar y eliminar pacientes. En este proyecto usamos para controlar el estado global la libreria de estados Zustand. Tambien usamos la libreria de alertas Toastify para avisar los registros agregados y eliminados.

TypeScript Vite React.JS Node.JS Tailwind
